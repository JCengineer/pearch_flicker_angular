Hello and thank you for taking the time to look at this mini application.
This app displays the latest 10 photos uploaded to the flickr website using their API. Each time the page is refreshed, a new set of photos are loaded.
It also display's advertisement banners in the slider. Each time the page is refreshed, the banner moves closer to the top of the list, so that on the 4th visit to the site, the banner is the first thing to be seen by the user.
This app is built using AngularJS, Flickr API, OwlCarousel, jQuery, Bootstrap, JavaScript, HTML5 and CSS. I also have experience using Ruby On Rails, PHP, C++ and more.
